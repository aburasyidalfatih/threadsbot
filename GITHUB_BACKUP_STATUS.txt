HOW TO PUSH TO GITHUB:

Option 1: Using HTTPS (Recommended for first time)
  1. Go to GitHub: https://github.com/aburasyidalfatih/threadsbot
  2. Create Personal Access Token:
     - Settings → Developer settings → Personal access tokens
     - Generate new token with 'repo' scope
  3. Run commands:
     git config --global credential.helper store
     git push -u origin main
  4. When prompted, use:
     - Username: aburasyidalfatih
     - Password: [paste your personal access token]

Option 2: Using SSH (More secure)
  1. Setup SSH key (if not already done):
     ssh-keygen -t ed25519 -C "your-email@example.com"
  2. Add public key to GitHub:
     - Settings → SSH and GPG keys → New SSH key
  3. Change remote:
     git remote set-url origin git@github.com:aburasyidalfatih/threadsbot.git
  4. Push:
     git push -u origin main
